# Simulador RSC-PCCTAE

Simulador do **Reconhecimento de Saberes e Competências (RSC-PCCTAE)** previsto no
**Decreto nº 13.048, de 3 de julho de 2026** (Lei nº 11.091/2005).

É uma página única (`index.html`), sem dependências e sem servidor: tudo roda no
próprio navegador. Nenhum dado é enviado para lugar nenhum.

## Como usar

1. Acesse a página (GitHub Pages) ou abra o arquivo `index.html` no navegador.
2. Selecione a **classe do cargo** e a sua **escolaridade**.
3. Informe a quantidade que você possui de cada item dos **Anexos I a VI**.
4. A pontuação, os critérios distintos e o **nível de RSC alcançado** são calculados na hora.
5. Use **Salvar respostas** para gerar um arquivo `.json` e **Carregar** para reabri-lo depois.

## Ressalva

Ferramenta pessoal, sem vínculo institucional, compartilhada a título meramente
informativo. O uso é facultativo e de responsabilidade exclusiva de cada usuário.
Os resultados são estimativas e **não vinculam, condicionam nem antecipam** qualquer
decisão da administração ou da CRSC-PCCTAE.
